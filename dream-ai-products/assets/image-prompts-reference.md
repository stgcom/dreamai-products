## Usage Notes

1. Copy the prompt text into your image generator (Nano Banana / DALL-E / Midjourney)
2. Use exact sizes as specified
3. After generating, add final text overlays in Canva/Figma for crisp typography
4. Brand consistency: always use #1B1464 background + #6C3CE9 accents
5. Export as PNG for best quality on all platforms

## Batch Generation Command (Nano Banana)

Once GEMINI_API_KEY is configured, run all prompts:

```bash
# Generate all LinkedIn cards
for prompt in li-01 li-02 li-03 li-04 li-05 li-06; do
  uv run /app/skills/nano-banana-pro/scripts/generate_image.py \
    --prompt "..." --filename "dream-ai-products/assets/images/$prompt.png" --resolution 1K
done
```

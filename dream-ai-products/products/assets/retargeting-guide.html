📊 Retargeting Pixel Setup Guide

Install tracking pixels to retarget visitors who don't convert. This turns one-time visitors into paying customers.

Why Retargeting Matters

Key stats:
  • Only 2% of website visitors convert on the first visit
  • Retargeted visitors convert 70% better than cold traffic
  • ROI on retargeting ads: 10x higher than standard display ads
  • It takes 7-13 touchpoints to convert a lead

1. Meta Pixel (Facebook + Instagram)

1

Create Meta Pixel

Go to: Facebook Business Manager → Events Manager → Pixels → Create

Copy your Pixel ID (format: 12-15 digit number)

2

Install Base Code

Add this to every page, right after <head>:

<!-- Meta Pixel Code --> <script> !function(f,b,e,v,n,t,s) {if(f.fbq)return;n=f.fbq=function(){n.callMethod? n.callMethod.apply(n,arguments):n.queue.push(arguments)}; if(!f._fbq)f._fbq=n;n.push=n;n.loaded=!0;n.version='2.0'; n.queue=[];t=b.createElement(e);t.async=!0; t.src=v;s=b.getElementsByTagName(e)[0]; s.parentNode.insertBefore(t,s)}(window, document,'script', 'https://connect.facebook.net/en_US/fbevents.js'); fbq('init', 'YOUR_PIXEL_ID'); fbq('track', 'PageView'); </script> <noscript><img height="1" width="1" style="display:none" src="https://www.facebook.com/tr?id=YOUR_PIXEL_ID&ev=PageView&noscript=1" /></noscript> <!-- End Meta Pixel Code -->
3

Track Custom Events

Add these event codes to track specific actions:

// Track Quiz Completion (scorecard) fbq('track', 'CompleteRegistration', {content_name: 'AI Readiness Quiz'}); // Track Calculator Use fbq('track', 'CalculateRevenue', {value: revenue_lost, currency: 'USD'}); // Track Product Purchase fbq('track', 'Purchase', {value: price, currency: 'USD'}); // Track Demo Booking fbq('track', 'Schedule', {content_name: 'AI Employee Demo'});

2. Google Analytics 4 (GA4) + Google Ads

1

Create GA4 Property

Go to: analytics.google.com → Admin → Create Property

Copy your Measurement ID (format: G-XXXXXXXXXX)

2

Install GA4 Tag

<!-- Google Analytics 4 --> <script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script> <script> window.dataLayer = window.dataLayer || []; function gtag(){dataLayer.push(arguments);} gtag('js', new Date()); gtag('config', 'G-XXXXXXXXXX'); </script> <!-- Google Ads Remarketing Tag (if running Google Ads) --> <script> gtag('config', 'AW-CONVERSION_ID'); </script>

💡 Pro Tip: Stacked Retargeting

Show different ads based on where they were in the funnel:

Day 1-3: Same product they viewed → Day 4-7: Testimonial/social proof → Day 8-14: Limited-time offer → Day 15-30: New content/lead magnet

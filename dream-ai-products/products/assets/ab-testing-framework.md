# Dream AI — A/B Testing Framework

## Testing Methodology

1. **Form hypothesis:** "Changing X will increase Y by Z%"
2. **Create variation** — only change ONE element per test
3. **Split traffic** 50/50
4. **Run for minimum** 2 weeks or 100 conversions (whichever comes last)
5. **Statistical significance:** 95% confidence level before declaring winner
6. **Declare winner**, implement, document learnings

## Agent Capabilities

### 1. AI Emergency Dispatch Agent
**Handles:** 24/7 inbound calls (especially after-hours emergencies)

**Flow:**
1. "Thanks for calling [Company]! I'm your AI dispatcher. Are you experiencing an emergency — no heat, no AC, or a leak?"
2. If emergency → capture address, problem description, callback number
3. Dispatch to on-call tech via SMS/app notification
4. If not urgent → schedule for next available slot
5. Caller gets confirmation text with tech ETA

**Priority Matrix:**
- 🔴 Emergency (no heat <40°F, gas leak, flooding) → On-call tech notified immediately
- 🟡 Urgent (AC out in summer, weird noises) → Next-day priority slot
- 🟢 Routine (maintenance, filter change, tune-up) → Normal scheduling

### 2. Seasonal Booking Agent
**Peak season:** Handles overflow when phone lines are slammed
**Pre-season:** Proactive outreach for maintenance plans
- "Hi [Name], it's almost summer! Want to schedule your AC tune-up before the rush?"
- "Winter is coming — when was the last time your furnace was inspected?"

### 3. Customer Follow-Up & Retention Agent
**Post-service:**
- "How's your AC running after yesterday's repair?"
- Rate satisfaction → Trigger review request if positive
- Flag issues → Alert dispatch for callback

**Maintenance reminders:**
- Monthly filter change reminders (if plan)
- Annual tune-up scheduling
- Warranty expiration alerts

### 4. Reputation Management Agent
**Triggers:** Post-service satisfaction check → Review request
**Google Business Profile:** Automated review solicitation
**Negative feedback:** Intercepted → Alert owner → Offer to resolve before public review

### 5. Lead Qualification (for new customers)
**Questions:**
1. What's the issue? (AC, heating, both)
2. Is the system completely down or partially working?
3. When did it start?
4. Is this a home or commercial property?
5. Do you have an existing service plan with us?

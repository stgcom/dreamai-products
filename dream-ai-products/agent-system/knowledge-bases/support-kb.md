## 2. Troubleshooting Guides

### 2.1 Voice Agent Issues

#### "The AI isn't answering calls"
**Steps:**
1. Check if the phone number is active and routing correctly (call it yourself)
2. Verify call forwarding is set up to the AI agent's line
3. Check the voice agent status in the dashboard → Settings → Agent Status
4. If using Twilio, verify the Twilio account has balance and is active
5. Check for any service alerts on the status page
6. If all checks pass, contact support with the date, time, and originating phone number

#### "Customers say the AI sounds robotic"
**Steps:**
1. Confirm the correct voice model is selected (Dashboard → Settings → Voice)
2. Check if the call quality issue is consistent or intermittent (could be network)
3. Review recent script changes — overly formal language can sound stiff
4. Adjust the voice settings (speed, pitch, emotion level)
5. If persistent, request a voice model update from the support team

#### "The AI isn't booking appointments correctly"
**Steps:**
1. Verify the calendar integration is connected (Dashboard → Integrations → Calendar)
2. Check calendar availability settings — are time slots actually open?
3. Review the booking script — ensure it's asking for all required info (name, email, phone, service)
4. Test the booking flow manually by calling the agent
5. Check the CRM integration — is the booking syncing?

#### "The AI is giving wrong information"
**Steps:**
1. Review the knowledge base content (Dashboard → Knowledge Base)
2. Update the FAQ entries with correct information
3. Check for conflicting entries (e.g., two different hours listed)
4. Clear the knowledge base cache (Settings → Knowledge Base → Refresh)
5. Retest by calling the agent with the specific question

### 2.2 Chat Agent Issues

#### "The chat widget isn't appearing on my website"
**Steps:**
1. Verify the embed code is correctly installed (check for missing quotes, wrong domain)
2. Clear your browser cache and test in incognito mode
3. Check if any ad blockers or browser extensions are blocking it
4. Verify your website's Content Security Policy allows our domain
5. Check the chat agent status in the dashboard (should show "Active")

#### "Chats aren't routing to my team"
**Steps:**
1. Check notification settings (Dashboard → Settings → Notifications)
2. Verify team email addresses and/or SMS numbers are correct
3. Check spam/junk folders for email notifications
4. Test with a manual handoff request
5. Verify the handoff rule is enabled in the chat flow settings

#### "The AI is giving generic responses"
**Steps:**
1. Check if the knowledge base has been populated (Dashboard → Knowledge Base)
2. Add specific FAQ entries related to your business
3. Update the system prompt with your brand voice and specific details
4. Clear the chat cache (Settings → Knowledge Base → Refresh)
5. Test with specific questions to verify the knowledge base is working

### 2.3 Integration Issues

#### "Contacts aren't syncing to my CRM"
**Steps:**
1. Check integration status (Dashboard → Integrations → [CRM Name])
2. Verify API keys/credentials are current
3. Check for error logs (Dashboard → Logs → Integration Errors)
4. Test with a manual contact sync
5. Verify field mapping (some CRMs require specific field names)

#### "Calendar sync is failing"
**Steps:**
1. Re-authenticate the calendar connection
2. Check for conflicting calendar permissions
3. Verify the calendar ID is correct
4. Test a manual booking and check if it appears on the calendar
5. Check if the calendar is set to the correct timezone

### 2.4 Billing Issues

#### "I was charged incorrectly"
**Steps:**
1. Check your invoice details (Dashboard → Billing → Invoices)
2. Review usage metrics for the billing period
3. Compare against your plan limits
4. Contact support with your invoice number and the specific discrepancy
5. We'll investigate and issue any credits within 48 hours

#### "My payment failed"
**Steps:**
1. Verify your card is valid and not expired
2. Check if your bank is blocking the charge (international/online transaction)
3. Update your payment method (Dashboard → Billing → Payment Methods)
4. Retry the payment
5. If persistent, contact support for alternative payment options

## 3. Refund Policy

### 3.1 Standard Policy
- **Setup fees:** Non-refundable once setup work has begun
- **Monthly fees:** Pro-rated refund available for the current month if service is cancelled within the first 14 days
- **Annual prepayment:** Pro-rated refund for unused months, minus a 15% early termination fee

### 3.2 Satisfaction Guarantee
> If, within the first 30 days, our AI agent does not:
> - Answer at least 90% of inbound calls/chats
> - Demonstrate clear value (booked appointments, captured leads, or time saved)
> 
> We will issue a **full refund of the first month's fee** (no setup fee refund if work was completed).
> 
> To qualify: you must have (1) provided all required setup information on time, (2) approved scripts within 48 hours of delivery, and (3) kept the service active for the full 30 days.

### 3.3 Dispute Resolution
1. Client submits a refund request via email (support@dreamai.com) or support chat
2. Support reviews the request within 2 business days
3. If approved, refund is processed within 5–7 business days to the original payment method
4. If denied, a detailed explanation is provided along with options (service adjustment, credits, etc.)
5. Appeals can be escalated to management (decision within 5 business days)

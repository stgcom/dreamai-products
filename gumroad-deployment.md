## 1. Gumroad Account Setup

### Step 1: Create Account
- Go to **gumroad.com** → **"Get Started"**
- Sign up with email (use: `products@dreamai.com` or founder's email)
- Choose **username:** `dreamai` (this becomes `gumroad.com/dreamai`)

### Step 2: Complete Profile
- **Display Name:** Dream AI
- **Bio:** "AI tools and templates that help small businesses stop losing money. Built by operators, not theorists."
- **Profile Picture:** Dream AI logo (square, min 400×400px)
- **Cover Image:** 1500×500px banner — dark navy with violet accent, tagline: "Your First AI Employee Starts Here"
- **Website:** dreamai.com
- **Social Links:** Add all active profiles (LinkedIn, Twitter/X, Instagram)

### Step 3: Payment Setup
- Go to **Settings → Payments**
- Connect **Stripe** (primary) — requires business details, bank account
- Connect **PayPal** (secondary) — optional but adds buyer trust
- Set payout schedule: **Weekly** (recommended for launch cash flow)
- **Tax info:** Complete W-9 (US) or equivalent — Gumroad handles sales tax collection

### Step 4: Store Branding
- **Store URL:** `gumroad.com/dreamai`
- **Store color accent:** Violet (#6C3CE9)
- **Store description:** "Plug-and-play AI tools for service businesses. Calculators, scripts, templates, and playbooks that pay for themselves in one use."
- **Custom domain (optional):** `shop.dreamai.com` — requires Gumroad Pro ($10/mo, skip for now)

### Step 5: Settings Checklist
- [ ] Email notifications ON for new purchases
- [ ] Fraud detection: ON (default)
- [ ] Allow tips: ON (for pay-what-you-want products)
- [ ] License keys: OFF (digital downloads, not software)
- [ ] Refund policy: 30-day, no-questions-asked

## 7. Analytics Setup

### 7A. Google Analytics on Gumroad Store

**Step-by-step:**
1. Go to **Google Analytics** → Admin → Data Streams → Web
2. Add your Gumroad store URL: `https://[username].gumroad.com`
3. Copy the **Measurement ID** (starts with `G-`)
4. In Gumroad: **Settings → Analytics → Google Analytics**
5. Paste the Measurement ID
6. Save and verify in GA4 Real-Time report (visit your store, confirm hit)

**Key events to track:**
- `page_view` — every product page view
- `begin_checkout` — when someone clicks "I want this"
- `purchase` — successful transaction

### 7B. Conversion Tracking Pixels

**Meta (Facebook/Instagram) Pixel:**
1. Go to **Meta Events Manager** → Pixels → Create Pixel
2. Copy Pixel ID
3. In Gumroad: **Settings → Analytics → Facebook Pixel**
4. Paste Pixel ID
5. Events tracked: `PageView`, `InitiateCheckout`, `Purchase`

**Google Ads Conversion Tracking:**
1. Go to **Google Ads → Tools → Conversions**
2. Create conversion action → "Website" → "Purchase"
3. Copy the conversion tag/snippet
4. In Gumroad: **Settings → Analytics → Custom tracking** (paste in header)
5. Set conversion value to pass dynamically (Gumroad supports this)

**TikTok Pixel (optional):**
1. Same flow as Meta — create pixel, get ID, paste in Gumroad custom tracking

### 7C. Revenue Dashboard Setup

**Option A: Gumroad Native Dashboard**
- **Gumroad → Analytics** shows: revenue, sales count, refund rate, top products, traffic sources
- Check daily during launch, weekly after
- Export CSV monthly for record-keeping

**Option B: Google Sheets Dashboard (recommended)**
1. Create a Google Sheet with tabs: Revenue, Products, Traffic, Emails
2. Weekly: Export Gumroad CSV → Paste into Revenue tab
3. Use formulas to track: weekly revenue, cumulative revenue, avg order value, conversion rate
4. Build a simple chart for visual tracking

- Use Gumroad webhook → N8N trigger
- On each sale: log to Google Sheet, send Slack/Discord notification, update CRM
